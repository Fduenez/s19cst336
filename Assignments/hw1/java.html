Java is one of the many programming languages out there. It was created in 1995 and owned by Oracle

Java is an object oriented, class based, concurrent, and to as few dependicies as possible.

You can write java once and run it anywhere. this means you can compile it as long as it is supported.

You can use Java for many different things like

  • Mobile applications (specially Android apps)
  • Desktop applications
  • Web applications
  • Web servers and application servers
  • Games
  • Many More!

As of now, Java is one of the most popular languages used for client-servers websites and has over billions of users.
